Move-In To Do list, started April 14, 2009

  • Build a small deck off the slider to meet code

Inside

Electrical

  • Install plugs in Emma’s room, Matt’s office
  • Install switch, light, switch plate, light box for second light, and second light in storage area
  • Build box for the entry light, improve the installation of the entry light
  • Build box for the kitchen light, install
  • Build boxes for dining room lights, install
  • Figure out how to put plates on the plugs behind the cabinets in the kitchen
  • Figure out how to put a plate on the freezer plug in the utility room

  • Install 220v plug for the dryer
  • Make sure the smoke detectors work
  • Install exterior electrical boxes, outlets, plugs above boxes
  • Install exterior light fixtures

Downstairs Bath

  • Pick out pedestal sink, buy sink, install
  • Buy faucet, plumbing connections, install
  • Tile shower installation (subbed out)
  • Buy shower fixtures, install
  • Buy flooring, install

Utility Room

  • Buy vent cover for freezer area, install in wall
  • Buy flooring, install
  • Buy door and pocket door, install
  • Buy and install the water heater

Kitchen

  • Buy counter, install
  • Install kitchen sink
  • Buy faucet, plumbing connections, install
  • Buy stove, install
  • Buy dishwasher, install
  • Figure out how to cover the island future hole in the floor in a more permanent fashion, build platform, skim coat the platform

Open area

  • Buy a brad nailer, wood for window trim
  • Install window trim on upper windows
  • Take down scaffolding because we no longer need it (yippee!)

Stairs

  • Fix bottom step so that it meets code for the appropriate amount of rise in each step
  • Install handrail, treads and balusters (subbed out, scheduled to start 6/29)

Landing/Upstairs Hall

  • Install flooring
  • Install bannisters, newel posts, balusters

Kids’ Bath

  • Pick out and buy the counter, install
  • Buy sinks, install
  • Buy shower fixtures, install
  • Buy faucets and plumbing connections, install
  • Buy flooring and underlayment, install
  • Buy toilet and install
  • Buy door and install

Master Bath

  • Tile shower installation (subbed out)
  • Buy shower fixtures, install
  • Buy flooring and underlayment, install
  • Pick out counter, buy and install
  • Buy faucet and plumbing connections, install
  • Buy toilet and install
  • Buy door and install

General Livability Improvements

  • Shelves in the pantry
  • Shelf in the linen/telecom closet for telecom/internet equipment
  • Rent floor cleaner and scrub floors
  • Permanent flooring in the upstairs bedrooms
  • Install knobs on cabinets
  • Finish installing oak flooring upstairs
  • Finish paint work in the master, Emma’s room, boy’s room, office, linen closet, kids’ bath, master bath
  • Buy pantry door and install

Telecom (not needed for move-in)

  • Get telephone inserts for telecom plates (currently backordered), install
  • Set up telephone switch box
  • Set up ethernet switch box/networking equipment
